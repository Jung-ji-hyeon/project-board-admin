# 어드민 서비스

게시판 서비스의 관리 도구를 개발해보는 프로젝트입니다. 

## 개발 환경

* Intellij IDEA Ultimate 2023.3.1
* Java 17
* Gradle 8.5
* Spring Boot 2.7.0

## 기술 세부 스택

Spring Boot

* Spring Boot Actuator
* Spring Data JPA
* Spring Security OAuth 2 Client
* Spring Security
* Thymeleaf
* Spring Web
* Spring WebSocket
* Lombok
* Spring Boot DevTools
* H2 Database
* MySQL Driver
* Spring Configuration Processor

그 외

* AdminLTE 3.2
* Heroku
